# cape bietet tiefe Unterstützung für einen bevorzugten Stack (TypeScript-Frontend + Rust-Backend)

## Outcome

cape gibt tiefe, meinungsstarke Unterstützung für ein **TypeScript-Frontend** und ein
**Rust-Backend**, während der generische Pfad für jeden anderen Stack vollständig offen
bleibt.

## Open points

- Was "tiefe Unterstützung" pro Schicht konkret bedeutet (Scaffolding, Konventionen,
  Stack-spezifische Review-Linsen, Test-Muster), muss noch festgelegt werden.
