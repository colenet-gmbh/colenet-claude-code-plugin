# cape offers deep support for a blessed stack (TypeScript frontend + Rust backend)

## Outcome

cape gives deep, opinionated support for a **TypeScript frontend** and a **Rust backend**,
while the generic path stays fully open for any other stack.

## Open points

- What "deep support" concretely means per layer (scaffolding, conventions, stack-specific
  review lenses, test patterns) still has to be pinned down.
