# cape enthält die wertvollen Teile des verworfenen superpowers/brainstorming-Skills

## Outcome

Die noch erhaltenswerten Teile des superpowers/brainstorming-Skills haben ihren Platz in
cape gefunden (oder wurden bewusst verworfen). Der Skill als Ganzes wurde fallengelassen,
weil er `/grill-with-docs` + `/feature` dupliziert hat.

## Realization job

Die vier Kandidaten-Teile bewerten und übernehmen, was seinen Platz verdient (können auch
keine sein):

- die Scope-Decompose-Heuristik,
- das harte Gate "kein Code vor einem freigegebenen Design",
- die Spec-Selbstreview-Checkliste,
- der Just-in-Time-Visual-Begleiter.

Jede Übernahme ist ein Issue dieses Features; alles aus einer externen Quelle Übernommene
wird gemäß `ATTRIBUTION.md` attribuiert.
