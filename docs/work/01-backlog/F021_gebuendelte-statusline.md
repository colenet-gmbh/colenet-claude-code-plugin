# Die gebündelte Statusline ist getestet, zuverlässig und wieder integriert

## Outcome

Die gebündelte Statusline ist getestet, funktioniert zuverlässig und ist wieder in README
und `/cape:setup` integriert — inklusive des Angebots, sie als Haupt-Statusline zu
installieren.

## Context

Die Statusline wurde aus README und `/cape:setup` herausgenommen, weil sie ungetestet war
und unklar ist, ob sie funktioniert — zudem schreibt sie in die globale
`~/.claude/settings.json`. Dies ist das V2-Feature: die Statusline sauber testen und
fertigstellen, bevor cape sie wieder aktiv anbietet.
