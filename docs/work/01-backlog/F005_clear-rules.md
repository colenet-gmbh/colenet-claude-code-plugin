# cape supports first-class, directive-style coding rules

## Outcome

cape has a first-class notion of terse, numbered directive rules (Spanier's `CC-`/`PL-`
style) that the skills consult while they work — so a repo's binding coding rules are found
and applied, not just prose in a doc.

## Realization job

Define the rule format and where rules live, and wire the skills (review, implement, tdd)
to consult them.
