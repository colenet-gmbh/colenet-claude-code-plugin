# Glossary

Shared vocabulary for the capd plugin. Keep these terms consistent across skills, the
README, and the docs.

## Main Flow

The guided capd path from an idea to a built slice:

```text
brainstorm → grill-with-docs → feature → split → build
```

"Workflow" is used as a synonym for the Main Flow. The skills that make it up are the
**workflow skills**; standalone skills that sit off it (e.g. `grill-me`, `ask-capd`) are
**utility skills**.

Do not call it a "spine" — that term is jargon and is not used in capd.
